# Setup Instructions

## JDK Version
This project uses JDK 21.

## How to Compile and Run

1. Clone the repository:
   git clone https://github.com/mxhaaroonx/LearnTrack.git

2. Open the project in IntelliJ IDEA

3. Make sure JDK 21 is configured:
   File → Project Structure → SDK → JDK 21

4. Run the project:
   Navigate to src/com/airtribe/learntrack/ui/Main.java
   Click the green play button or right click → Run 'Main'
